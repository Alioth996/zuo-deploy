## v0.3.2 的缺点

- 在一个 linux 服务器只能运行在一个项目下，多项目会有问题，仅支持执行当前目录下 deploy-master.sh 脚本
- 不支持修改服务器 nginx 配置
- 不支持实时修改 shell 脚本。需要先在项目中修改在提交 git，服务器拉取再次执行才能生效
